# Military Security Science Popularization Website

An interactive web-based science popularization platform designed to introduce the fundamentals of military security through data visualization, interactive learning, and responsive web design.

## Preview

> *(You can place screenshots here later.)*

## Features

- 📊 Interactive data visualization with Chart.js
- 🎴 Flip-card introduction to different dimensions of military security
- 🌍 Global military security trends
- 📝 Knowledge quiz with instant feedback
- 🧩 Scenario-based interactive exercises
- ⚖️ National security laws and policy overview
- 📱 Responsive layout for desktop and mobile devices

## Technologies

- HTML5
- CSS3
- JavaScript (ES6)
- Chart.js
- Font Awesome

## Project Structure

```
index.html
```

All styles, scripts and page content are integrated into a single HTML file for easy deployment and demonstration.

## Purpose

This project was developed as a course assignment to explore modern front-end web development techniques while presenting military security knowledge in a more engaging and interactive way.
